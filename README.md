This is a weboage with local storage capabilities, allowing the user to create a very simple lists of custom tasks that will stay between refreshes.
Some of the mahor included functions are search filtering, cutsom tags, task deletion, and task creation.

This was my first project with Local Storage, so learning assists such as AI and various online coding examples were used to help me understand how to create and structure this project.
